# CET-4/CET-6考试时间模拟器

CET-4-practice-clock：一个用于模拟大学英语四级(CET-4)和六级(CET-6)考试时间流程的Web应用程序。

---

## 项目简介

此项目是一个基于Web的CET-4/CET-6考试时间模拟器，帮助考生熟悉考试流程和时间安排。模拟器严格按照CET-4和CET-6考试的时间安排设计，包括各个考试环节的倒计时和实时时间显示。

## 功能特点

- 🎓 完整模拟CET-4和CET-6考试流程（2025年12月13日考试）
- ⏱️ 实时显示考场时间和考试倒计时
- 📋 可视化展示所有考试环节及其状态（待开始/进行中/已完成）
- ▶️ 支持开始、暂停、重置和跳转到指定环节等功能
- 📊 进度条显示考试整体进度
- 🔁 支持在CET-4和CET-6之间切换
- 📱 响应式设计，适配不同设备屏幕

## 响应式设计

本项目采用响应式设计，能够在不同尺寸的设备上提供最佳的用户体验：

- **小屏幕设备**（屏幕宽度 ≤ 800px）：
  - 采用单栏布局，内容区域固定为400px宽度并居中显示
  - 适合手机和平板等移动设备使用

- **大屏幕设备**（屏幕宽度 > 800px）：
  - 自动切换为两栏布局，总内容宽度为800px
  - 左侧显示主要控制面板和实时信息
  - 右侧显示考试流程时间线
  - 超出部分保持白色背景，整体内容居中显示

这种设计确保了在各种设备上都能获得良好的可视性和操作体验。

## 考试流程

### CET-4考试流程（上午9:00-11:20）
1. 考前准备 (9:00-9:10)
2. 写作 (9:10-9:40)
3. 听力 (9:40-10:05)
4. 收答题卡1 (10:05-10:10)
5. 阅读理解+翻译 (10:10-11:20)

### CET-6考试流程（下午15:00-17:25）
1. 考前准备 (15:00-15:10)
2. 写作 (15:10-15:40)
3. 听力 (15:40-16:10)
4. 收答题卡1 (16:10-16:15)
5. 阅读理解+翻译 (16:15-17:25)

## 如何在本地运行

### 方法一：直接用浏览器打开（推荐）
1. 在文件资源管理器中找到项目文件夹
2. 双击 `index.html` 文件，它将在默认浏览器中打开
3. 或者右键点击 `index.html` 文件，选择"打开方式" -> 选择你喜欢的浏览器

### 方法二：使用Live Server（VS Code用户）
1. 在VS Code中打开项目文件夹
2. 安装"Live Server"扩展（如果尚未安装）
3. 右键点击 `index.html` 文件
4. 选择"Open with Live Server"
5. 浏览器将自动打开并运行项目

### 方法三：使用Python简单HTTP服务器
1. 打开终端或命令提示符
2. 切换到项目文件夹目录
3. 运行以下命令：
   ```
   python -m http.server 8000
   ```
4. 在浏览器地址栏输入 `http://localhost:8000` 并回车

### 方法四：使用Node.js http-server
1. 安装Node.js（如果尚未安装）
2. 在终端中运行 `npm install -g http-server`
3. 在项目文件夹中运行 `http-server`
4. 在浏览器中访问显示的地址（通常是 `http://localhost:8080`）
