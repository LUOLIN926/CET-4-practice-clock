# CET-4/CET-6考试时间模拟器

CET-4-practice-clock：一个用于模拟大学英语四级(CET-4)和六级(CET-6)考试时间流程的Web应用程序。

## 项目简介

此项目是一个基于Web的CET-4/CET-6考试时间模拟器，帮助考生熟悉考试流程和时间安排。模拟器严格按照CET-4和CET-6考试的时间安排设计，包括各个考试环节的倒计时和实时时间显示。此外，还支持自定义考试配置，允许用户创建和管理自己的考试安排。

## 功能特点

- 🎓 完整模拟CET-4和CET-6考试流程（2025年12月13日考试）
- ⏱️ 实时显示考场时间和考试倒计时
- 📋 可视化展示所有考试环节及其状态（待开始/进行中/已完成）
- ▶️ 支持开始、暂停、重置和跳转到指定环节等功能
- 📊 进度条显示考试整体进度
- 🔁 支持在CET-4和CET-6之间切换
- 📱 响应式设计，适配不同设备屏幕
- ⚙️ 自定义考试配置功能，支持创建和管理个性化考试安排

## 详细功能介绍

### 1. 考试模拟功能
- **完整考试流程模拟**：严格按照官方考试时间安排，模拟完整的CET-4（上午9:00-11:20）和CET-6（下午15:00-17:25）考试流程
- **实时考场时间显示**：显示模拟的考场实时时间，让考生清楚当前考试时间点
- **考试倒计时**：显示距离考试结束的剩余时间
- **环节倒计时**：显示当前考试环节的剩余时间
- **考试状态追踪**：实时显示当前所处的考试环节及其描述信息

### 2. 考试环节可视化
- **环节状态标识**：每个考试环节都有明确的状态标识（待开始/进行中/已完成）
- **环节详情展示**：显示每个环节的名称、持续时间、描述和对应的考场时间
- **进度条显示**：通过彩色进度条直观展示整体考试进度
- **统计数据面板**：显示总考试时间、剩余时间、已完成环节数等关键统计数据

### 3. 控制功能
- **开始/继续考试**：启动考试计时器，支持从中断点继续考试
- **暂停考试**：暂停计时器，保留当前考试状态
- **重置考试**：将考试状态重置为初始状态
- **再次考试**：考试结束后可以重新开始一次新的考试
- **跳转到指定环节**：通过下拉菜单快速跳转到任意考试环节
- **跳转到下一环节**：一键跳转到下一考试环节，方便测试或演示

### 4. 考试类型切换
- **CET-4/CET-6切换**：通过切换按钮可以在CET-4和CET-6考试模式之间自由切换
- **自定义考试模式**：支持创建和使用自定义考试配置
- **自动配置更新**：切换考试类型时，所有相关配置（时间、环节等）会自动更新

### 5. 自定义考试功能
- **创建自定义考试**：用户可以通过专门的配置页面创建自己的考试安排
- **考试环节管理**：支持添加多个考试环节，设置每个环节的名称、时长和描述
- **考试时间设置**：可以设置考试的开始时间和日期
- **配置保存和管理**：支持保存多个自定义考试配置，并可以随时应用或删除
- **跨日期考试支持**：正确处理跨越午夜的考试时间（如23:00到次日00:20）

### 6. 用户界面特性
- **响应式设计**：
  - 小屏幕设备（≤800px）：单栏布局，内容区域400px居中显示
  - 大屏幕设备（>800px）：双栏布局，总宽度800px，左侧控制面板，右侧考试流程时间线
- **可折叠头部信息**：支持隐藏/显示顶部考试信息区域，提供更多屏幕空间
- **直观的状态指示器**：使用不同颜色标识不同状态（进行中-红色，待开始-蓝色，已完成-绿色）
- **友好的交互设计**：所有按钮都有悬停效果，提供良好的用户体验

### 7. 数据持久化
- **状态保存**：利用浏览器本地存储技术，在页面刷新后能恢复之前的考试状态
- **配置保存**：自定义考试配置会保存在本地存储中，方便下次使用
- **状态恢复**：即使意外刷新页面，也能继续之前的考试进程

## 响应式设计

本项目采用响应式设计，能够在不同尺寸的设备上提供最佳的用户体验：

- **小屏幕设备**（屏幕宽度 ≤ 800px）：
  - 采用单栏布局，内容区域固定为400px宽度并居中显示
  - 适合手机和平板等移动设备使用

- **大屏幕设备**（屏幕宽度 > 800px）：
  - 自动切换为两栏布局，总内容宽度为800px
  - 左侧显示主要控制面板和实时信息
  - 右侧显示考试流程时间线
  - 超出部分保持白色背景，整体内容居中显示

这种设计确保了在各种设备上都能获得良好的可视性和操作体验。

## 如何在本地运行

### 方法一：直接用浏览器打开（推荐）
1. 在文件资源管理器中找到项目文件夹
2. 双击 `index.html` 文件，它将在默认浏览器中打开
3. 或者右键点击 `index.html` 文件，选择"打开方式" -> 选择你喜欢的浏览器

### 方法二：使用Live Server（VS Code用户）
1. 在VS Code中打开项目文件夹
2. 安装"Live Server"扩展（如果尚未安装）
3. 右键点击 `index.html` 文件
4. 选择"Open with Live Server"
5. 浏览器将自动打开并运行项目

### 方法三：使用Python简单HTTP服务器
1. 打开终端或命令提示符
2. 切换到项目文件夹目录
3. 运行以下命令：
   ```
   python -m http.server 8000
   ```
4. 在浏览器地址栏输入 `http://localhost:8000` 并回车

### 方法四：使用Node.js http-server
1. 安装Node.js（如果尚未安装）
2. 在终端中运行 `npm install -g http-server`
3. 在项目文件夹中运行 `http-server`
4. 在浏览器中访问显示的地址（通常是 `http://localhost:8080`）
